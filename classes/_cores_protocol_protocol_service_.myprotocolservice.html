  • Implementation of ProtocolService.fromURL

    • Defined in src/cores/protocol/protocol-service.ts:263

    from string url, transform to protocol-param. mode is dependent on body condition.

    • if body is undefined, then mode will be 'GET'
    • if body is not undefined, then mode will be 'POST'.

    Parameters

    • context: NextContext

      the current execute context via controller.

    • url: string

      url string must start with 'lemon://' like lemon://lemon-hello-api/hello/0, or 'api://'

    • Optional param: ProtocolBody

    • Optional body: ProtocolBody
